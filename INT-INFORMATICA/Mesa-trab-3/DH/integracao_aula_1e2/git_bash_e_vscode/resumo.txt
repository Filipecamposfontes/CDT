Resumo: O git-bash é uma aplicação para
 Windows que emula a experiência de utilizar
  o shell do Linux ou Osx, além de conter
   todos os comandos Git. 
   Neste breve artigo mostrarei como modificar
    o terminal integrado do vscode, atribuindo
     o git-bash. No entanto o passo a passo 
     a seguir também pode ser utilizado para
      definir o Powershell como terminal padrão,
       uma vez que o vscode vem com o cmd.exe configurado por padrão.
